# 二分法搜索

basic
---
左右边界，取中位数查看

**防止溢出**：mid = l + (r - l) // 2

```
while L <= R:  # 在L > R时停止
    if mid == target:
        R = mid
    if mid < target:
        L = mid + 1 # 左边界收缩
    else:
        R = mid

```
